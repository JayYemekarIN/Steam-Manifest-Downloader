# Steam Manifest Downloader

A Node.js utility to assist in downloading Steam manifests.

## Installation

1. Clone the repository:
```bash
git clone [https://github.com/JayYemekarIN/Steam-Manifest-Downloader.git](https://github.com/JayYemekarIN/Steam-Manifest-Downloader.git)
```

2. Navigate to the project directory:
```bash
cd Steam-Manifest-Downloader
```

3. Install the required dependencies:

```bash
npm init -y && npm i express axios
```
## Usage

You can start the application using one of the following methods:

### Option 1: NPM Command

Run the start script defined in ```package.json:```
```bash
npm run start
```
### Option 2: Windows Batch File
Simply double-click or run the included batch file in your terminal:
```bash
app.bat
```
The server will start on port 3000 (default) and it automatically opens the browser:
*http://localhost:3000*
